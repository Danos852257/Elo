This site is a decision making app that simplifies complicated decisions into 2 choices

HOW TO USE
Create an account and you can start making rankings
Click on New Ranking and fill out the name and competitors field
Each new line is a separate competitor, so don't put them on the same line with commas
Put them on different lines
Then click on the ranking and choose whichever option you think is better
The Elo algorithm determines the rankings
Learn more at https://en.wikipedia.org/wiki/Elo_rating_system#Mathematical_details

You can use the app for pretty much anything, including favorite color, book, major, college, movie, etc

WARNING: Do not store any personal information on this website. It was not made with security in mind.
